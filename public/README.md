# Espace public

Ce dossier est réservé aux documents publics et publiables.

Peuvent y figurer :

- documents déjà publics ;
- tracts validés pour diffusion ;
- fiches expurgées ;
- ressources dont la publication ne révèle aucune information interne, nominative ou confidentielle.

Ne jamais y placer :

- accord INEOS complet ;
- règlement interne complet ;
- BDESE ;
- document nominatif ;
- document RH, médical, disciplinaire ou confidentiel.

Le dossier `documents/` du site public obéit aux mêmes règles.
