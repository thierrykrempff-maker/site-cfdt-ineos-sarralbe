# Espace privé versionnable

Ce dossier contient uniquement des références, maquettes, métadonnées et fiches sans contenu sensible.

Il peut contenir :

- modules CFDT Nexus en développement ;
- fiches expurgées ;
- références documentaires sans reproduction d'accord complet ;
- données de démonstration fictives.

Il ne doit jamais contenir :

- accord INEOS complet ;
- règlement interne complet ;
- BDESE ;
- données nominatives ;
- documents RH, médicaux, disciplinaires ou confidentiels.

Les documents sensibles doivent rester hors dépôt.
