# CFDT Nexus - Bibliothèque documentaire

Sprint V1 - Bibliothèque documentaire.

## Fonctionnalités

- page Bibliothèque directement utilisable ;
- ajout de document avec backend simulé par `localStorage` ;
- classement par catégorie ;
- moteur de recherche instantané ;
- filtres par catégorie et confidentialité ;
- fiche détaillée d'un document ;
- export JSON ;
- données préparées pour indexation IA, recherche sémantique, Routeur Intelligent et alimentation des Agents IA.

## Catégories V1

- Convention collective
- Accord d'entreprise
- Règlement intérieur
- Jurisprudence
- Modèle de courrier
- PV CSE

## Données documentaires

Chaque document contient titre, catégorie, description, mots-clés, date, version et niveau de confidentialité.

## Limite V1

Les fichiers ne sont pas encore téléversés. La V1 stocke les références et les métadonnées localement dans le navigateur.
