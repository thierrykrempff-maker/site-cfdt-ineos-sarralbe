# CFDT INEOS Sarralbe

## Priorite V1 de CFDT Nexus

Le module central de CFDT Nexus est le Document Intelligence Center.

La V1 est recentree sur trois priorites :

1. Analyse documentaire intelligente.
2. Generation de questions CSE / CSSCT.
3. Preparation des actions syndicales a partir des documents.

Les modules Dossiers d'accompagnement et Bibliotheque documentaire restent importants, mais ils servent d'appui au DIC : organiser les situations, retrouver les references utiles et transformer un document en questions, points de vigilance et actions concretes.

## Reference principale

Le document [MANIFESTE_CFDT_NEXUS.md](docs/MANIFESTE_CFDT_NEXUS.md) est la reference principale du projet.

Avant toute evolution importante :

- verifier que la demande respecte le manifeste ;
- si une demande est contraire au manifeste, expliquer pourquoi ;
- proposer une alternative conforme ;
- citer le principe du manifeste concerne lorsque c'est pertinent.

Les decisions d'architecture et de developpement doivent rester coherentes avec les principes du manifeste :

- etre proche des salaries ;
- expliquer clairement ;
- defendre sans diviser ;
- agir avec serieux, transparence et determination ;
- informer, accompagner, alerter si necessaire et proposer des solutions concretes.

## Securite documentaire

Les accords INEOS complets, reglements internes, donnees nominatives, documents BDESE et pieces confidentielles ne doivent jamais etre stockes dans ce depot ni exposes sur le site public.

La reference de securite documentaire est [SECURITY_POLICY.md](knowledge/SECURITY_POLICY.md).
