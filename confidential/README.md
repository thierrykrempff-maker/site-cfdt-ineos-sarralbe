# Espace confidentiel

Ce dossier doit rester vide.

Il existe uniquement comme garde-fou documentaire pour rappeler qu'aucun document confidentiel ne doit être stocké dans ce dépôt.

Ne jamais placer ici :

- accord INEOS complet ;
- règlement interne complet ;
- BDESE ;
- document nominatif ;
- courrier RH individuel ;
- document disciplinaire ;
- document médical ;
- pièce confidentielle CSE ou CSSCT.

Les documents sensibles sont analysés uniquement ponctuellement, hors dépôt public, avec validation humaine.
